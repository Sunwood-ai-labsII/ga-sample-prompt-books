# Prompt Books

Prompt Booksは、プロンプトを閲覧・検索できるウェブアプリケーションです。

## 機能

-   プロンプトの一覧表示
-   タイトルや説明文でのプロンプト検索

## 使い方

1.  `index.html`をウェブブラウザで開きます。
2.  プロンプトの一覧が表示されます。
3.  検索ボックスを使ってプロンプトを絞り込めます。

## ディレクトリ構造

```
.
├── index.html
├── style.css
└── script.js
```
